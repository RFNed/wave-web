# Wave-Web
### Веб-сайт проекта Wave и его Бэкенд.

Для полной настройки проекта потребуется скачать зависимости **Python** в `requierements.txt`

---
### Настройте конфигурацию 
Переименуйте файл `.env.example` в корневой папке проекта и такой же файл внутри папки `website` в `.env` (в обоих случаях)


Заполните все требуемые данные в `.env` файлах, и запустите фронтенд
```bash
cd website
npm i # можно один раз вызвать
npm run dev
```

Так же запустите бэкенд
```bash
uvicorn backend:app --reload
```

Готово! Вы можете вносить свои правки в проект и пуллить в репозиторий!
